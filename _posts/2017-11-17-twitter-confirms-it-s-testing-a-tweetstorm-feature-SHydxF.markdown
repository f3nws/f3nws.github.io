As if upping its character limit to 280 wasn't enough, Twitter is testing a dedicated feature to make it easier for users to tweetstorm. The test first surfaced in September, but Twitter is now confirming the experiment, which is currently live only to people who are part of Twitter's Experiments Program . SEE ALSO: Twitter’s 280-character limit is exactly what tweets didn’t need The latest version of the feature, which was spotted by Android Police, allows users to add multiple tweets to a single thread using a new "+" icon in the tweet composer. It's not clear if it limits you to a specific number of tweets, though it appears each one has the same 280-character limit as the rest of Twitter. Read more...
